Page Transition Animation

This page uses a sliding transition effect to create a smooth and dynamic user experience. The transition is achieved by animating the `clip-path` property, which initially hides the page and gradually reveals it from right to left. As the animation progresses, the content smoothly appears, providing a seamless visual flow between pages. This enhances navigation, making the user interaction feel more fluid and engaging.
